## How to publish on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under *Source*, select `main` branch and `/ (root)`
4. Save — the site will be live at `https://<your-username>.github.io/Portfolio`

No build step needed. All pages are plain HTML with Chart.js loaded from CDN.
